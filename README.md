## Featured Projects

### [Cgl Regulation Explorer](https://github.com/gaodandan-ai/cgl_regulation)

An interactive regulatory-metabolic analysis platform for *Corynebacterium glutamicum*.  
It integrates transcription factor and sRNA regulatory evidence with genome-scale metabolic models, enzyme constraints, pathway-level analysis, topology annotation, and AI-assisted literature summaries.

Live demo: https://cgl-regulation.vercel.app/

**Keywords:** regulatory networks, metabolic modeling, Cytoscape.js, RF scoring, FBA/ecFBA, RAG, microbial systems biology

### [Whole-Genome Resequencing Pipeline](https://github.com/gaodandan-ai/cg_sequencing)

A reproducible pipeline for *Corynebacterium glutamicum* whole-genome resequencing analysis, including FASTQ quality control, trimming, BWA alignment, variant calling, allele-frequency filtering, snpEff annotation, and Excel report generation.

**Keywords:** WGS, variant calling, BWA, samtools, bcftools, snpEff, reproducible genomics

### [CRISPRi Thermal Screening Pipeline](https://github.com/gaodandan-ai/CRISPRi_thermal)

An automated analysis pipeline for CRISPR interference screening under thermal stress, supporting both single-gene and dual-gene sgRNA library workflows.

**Keywords:** CRISPRi, sgRNA screening, thermal stress, bowtie2, count matrix generation

### [GP960 Growth Screen Pipeline](https://github.com/gaodandan-ai/GP960-growth-screen-pipeline-Red)

A Python pipeline for high-throughput microbial growth curve screening data generated by the GP960 system, including OD table standardization, curve cleaning, fitness metric calculation, and visual summaries.

**Keywords:** growth curves, high-throughput screening, stress tolerance, Python, data analysis
